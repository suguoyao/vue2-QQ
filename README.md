# 基于Vue2.0高仿手机QQ的单页面应用

## 概述

这是一个使用Vue2.0来模仿手机QQ的WebApp，出于个人兴趣和对Vue的喜爱写下这个项目，目前处于初步开发阶段。

有什么建议和问题欢迎大神们在[Issues](https://github.com/suguoyao/vue2-QQ/issues)提出，感兴趣的朋友给个充满动力的star吧！

**持续更新中**

最近更新时间：2018-03-31
